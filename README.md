# Pi_Surveilr
Motion detection using Raspberry Pi and USB Webcam with one or two cameras.
# Features  
Notify user via Telegram when motion is detected.  
Send images to Dropbox (From PyImageSearch)
# Working on..
Send images to other cloud storages.
